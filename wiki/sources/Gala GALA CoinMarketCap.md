## Summary

CoinMarketCap listing page for $GALA with ecosystem description from Gala. Covers the GalaChain L1 blockchain, Founder's Node DePIN infrastructure, tokenomics (no ICO, dynamic emission, halving tranches, burn), and security ratings.

## Key Points

- Live price: $0.002845; market cap ~$135M; ranking #158.
- Max supply: 50B GALA; circulating ~47.6B.
- Daily emission: halving based on total supply thresholds (tranches from 68M/day down to <1M/day).
- No ICO: all GALA enters circulation via Founder's Node operator rewards.
- Burn mechanism: GALA used as gas is burned.
- Founder's Nodes: 50,000 total; 50/50 split of daily rewards between operators and Gala (Conservatorial Entity).
- Founded 2019 by Eric Schiermeyer (Zynga co-founder).
- CertiK rating: #3 most secure altcoin; AAA/Gold badge.
- GalaChain supports bridging to Ethereum, Binance Smart Chain.
- External developers can use GalaChain SDK and Creator Portal.
