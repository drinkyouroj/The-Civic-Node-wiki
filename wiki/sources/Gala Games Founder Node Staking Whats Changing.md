## Summary

Official Gala announcement of the Founder Node Staking program launching June 16, 2025. The program ties daily Founder Node rewards to $GALA holdings (and later locked stake). A new token ($GSTAKE) is introduced. The 20,000-node minimum online threshold is removed. Four-phase rollout planned through Q3 2025.

## Key Points

- Starting June 16, 2025: Founder Node rewards scale based on $GALA held in GalaChain wallet.
- Full rewards require 1 million $GALA per node; partial rewards for less.
- New token $GSTAKE introduced: initially 1:1 with $GALA; minted by converting eligible NFTs.
- 20,000-node minimum threshold removed — daily emissions continue regardless of node count.
- Four phases: (1) Holdings-based rewards [Jun 16]; (2) $GSTAKE/NFT exchange [late Jun]; (3) Locking $GALA or $GSTAKE [Q3]; (4) $GSTAKE lending program [Q3].
- $GSTAKE is a GalaChain native token; transferable and tradeable on Gala platforms.
- Phase 4: $GSTAKE lending — surplus holders can lend to other node operators for a share of rewards.

## Newsletter Angles

- The staking upgrade solves Gala's "dead weight" problem: node operators who weren't actually using the network were still earning rewards. By requiring $GALA holdings, Gala ties economic participation to network commitment.
- $GSTAKE as a second-layer token: introducing a governance/staking token on top of the existing $GALA creates a two-token system with different liquidity and lockup dynamics — a sophisticated but fragile architecture.
- The NFT-to-$GSTAKE conversion mechanism (Phase 2) creates an interesting cross-market: existing NFT holders can convert to staking eligibility, effectively making NFTs a gateway to yield.
